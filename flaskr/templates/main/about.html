What is this site?

This is a fully functional personal hobby project built with Python Flask.
It consists several common key components of modern web applications

  • A web framework to support business logics (in this case allow user to register and post blogs)
  • A remote server to handle http requests
  • Embedded use of popular APIs in the backend
